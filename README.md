# 🫀 Qalb (قلب) - Heart Wellness & Spiritual Balance

[![Firebase](https://img.shields.io/badge/Firebase-FFCA28?style=for-the-badge&logo=firebase&logoColor=black)](https://firebase.google.com)

**Qalb** is a premium, holistic health application designed to bridge the gap between modern medical science and Islamic spiritual practices. By combining Machine Learning-driven heart attack risk prediction with traditional Islamic wellness modules, Qalb provides a comprehensive approach to "Healing the Heart" in both a physical and spiritual sense.

---

## ✨ Key Features

### 🩺 Physical Health & Prevention
- **Heart Attack Risk Prediction:** Advanced ML-integrated assessment to evaluate cardiovascular health risks.
- **Smoking Cessation Tracker:** Dedicated module to help users quit smoking with progress tracking.
- **Breathing Exercises:** Guided breathing routines for stress reduction and heart health.
- **Health Analytics:** Dynamic charts and visualizations of your health metrics using `fl_chart`.

### 🌙 Islamic Wellness
- **Dhikr Counter:** Integrated Tasbih with streak tracking to maintain spiritual consistency.
- **Sunnah Nutrition:** A curated guide to traditional foods recommended in the Sunnah and their scientific health benefits.
- **Fasting Tracker:** Manage and track your voluntary and mandatory fasts with historical data.
- **Prayer Times & Hijri Calendar:** Accurate prayer timings and date tracking based on your location.
- **Nasiha Chat:** AI-powered spiritual guidance and Quranic wisdom for mental and emotional well-being.

### 🎨 Premium User Experience
- **Modern UI/UX:** A stunning, glassmorphic design language with soft gradients and elegant typography.
- **Dark Mode Optimized:** Fully optimized for low-light environments with a sophisticated emerald theme.
- **Seamless Animations:** Smooth transitions and micro-interactions using Lottie and custom Flutter animations.

## 🚀 Tech Stack

- **Framework:** [Flutter](https://flutter.dev) (v3.0+)
- **State Management:** [Riverpod](https://riverpod.dev) with Code Generation
- **Backend:** [Firebase](https://firebase.google.com) (Auth, Firestore, Storage)
- **Database:** [Hive](https://pub.dev/packages/hive) (Local persistence)
- **AI Integration:** [Google Generative AI](https://pub.dev/packages/google_generative_ai) (Gemini)
- **Navigation:** [GoRouter](https://pub.dev/packages/go_router)
- **UI Components:** [Google Fonts](https://fonts.google.com), [Lottie](https://lottiefiles.com), [Shimmer](https://pub.dev/packages/shimmer)

## 🛠️ Getting Started

### Prerequisites
- Flutter SDK installed on your machine.
- Firebase project configured (add `google-services.json` and `GoogleService-Info.plist`).

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/qalb.git
   ```
2. Navigate to the project directory:
   ```bash
   cd qalb
   ```
3. Install dependencies:
   ```bash
   flutter pub get
   ```
4. Run code generation:
   ```bash
   flutter pub run build_runner build --delete-conflicting-outputs
   ```
5. Run the application:
   ```bash
   flutter run
   ```
